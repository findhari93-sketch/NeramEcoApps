Design Section 2: Routes & Navigation

Where the Course Plan lives in Nexus for teachers vs students

Teacher Routes

📂 apps/nexus/src/app/(teacher)/course-plans/
Pages
/teacher/course-plans
  → List all plans for this classroom
  → Card per plan: name, status, progress %
  → "Create Plan" button

/teacher/course-plans/[planId]
  → Plan overview dashboard
  → Tabs: Overview | Weeks | Homework | Tests | Drill | Resources

/teacher/course-plans/[planId]/weeks/[weekId]
  → Week detail: all sessions for that week
  → Assign dates, push to timetable
  → "Push entire week" bulk action

/teacher/course-plans/[planId]/homework
  → Homework grid (like Teams screenshot)
  → Columns: assignments, Rows: students
  → Click cell → review submission
Key Actions
Create Plan
Dialog: name, classroom, duration, schedule pattern, teaching team
Add/Edit Session
Pick topic, assign teacher, write description, attach homework
Push to Timetable
Pick date/time → creates scheduled_class + Teams meeting → updates session status to "scheduled"
Grade Homework
View submission, see attachments, give points + feedback, mark reviewed/returned
Manage Drill Questions
Add/edit/reorder the repeat questions list

Student Routes

📂 apps/nexus/src/app/(student)/course-plan/
Pages
/student/course-plan
  → Active course plan for their classroom
  → "Today's Plan" hero card at top
  → Week tabs below (Week 1-5)
  → Quick links: Homework | Drill | Tests

/student/course-plan/homework
  → All homework with submission status
  → Filters: pending / submitted / reviewed
  → Upload photos, text responses

/student/course-plan/drill
  → Flashcard-style repeat questions
  → Flip to reveal answer
  → Self-mark: "Got it" / "Need practice"

/student/course-plan/tests
  → Upcoming & past weekly tests
  → Links to QB test system for taking tests
  → Score history per week

API Routes

apps/nexus/src/app/api/course-plans/
Plan CRUD
GET  /api/course-plans → list plans
POST /api/course-plans → create plan
GET  /api/course-plans/[id] → get plan detail
PUT  /api/course-plans/[id] → update plan

Sessions
GET  /api/course-plans/[id]/sessions → list sessions
POST /api/course-plans/[id]/sessions → create/update session
POST /api/course-plans/[id]/sessions/push → push to timetable
POST /api/course-plans/[id]/sessions/push-week → bulk push week
Homework
GET  /api/course-plans/[id]/homework → list homework
POST /api/course-plans/[id]/homework → create homework
GET  /api/course-plans/[id]/homework/grid → grading grid
POST /api/course-plans/homework/[hwId]/submit → student submit
POST /api/course-plans/homework/[hwId]/review → teacher review

Drill, Tests, Resources
GET/POST /api/course-plans/[id]/drill → manage drill Qs
GET/POST /api/course-plans/[id]/tests → manage tests
GET/POST /api/course-plans/[id]/resources → manage links
