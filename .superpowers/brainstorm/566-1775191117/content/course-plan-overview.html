Data Model

Database Entities & Relationships
── Plan Structure ──
course_plans
  ├── name, description, classroom_id
  ├── duration_weeks, schedule_pattern
  ├── status (draft/active/completed)
  └── teaching_team (JSON)

course_plan_weeks
  ├── week_number, title, goal
  └── plan_id (FK)

course_plan_sessions
  ├── week_id, day_of_week, slot (am/pm)
  ├── topic_id (FK → nexus_topics)
  ├── teacher_id, title, description
  ├── scheduled_class_id (FK, nullable)
  └── status (planned/scheduled/completed)
── Content & Tracking ──
course_plan_homework
  ├── session_id, title, description
  ├── type (drawing/mcq/study/review)
  └── due_date, estimated_minutes

homework_submissions
  ├── homework_id, student_id
  ├── attachments (files/photos)
  └── status, teacher_feedback

course_plan_tests
  ├── plan_id, week_number
  ├── title, question_count, duration_min
  └── linked to QB test system

course_plan_resources
  ├── topic_id OR session_id
  └── title, url, type
── Quick Drill ──
course_plan_drill_questions
  ├── plan_id, question_text, answer
  ├── frequency_note, sort_order
  └── Standalone widget (not in QB)

Teacher vs Student Views

👩‍🏫 Teacher View
Course Plan Builder
Create/edit weeks, sessions, assign topics & teachers
Week Scheduler
Assign dates, push to timetable (single or bulk), auto-create Teams meetings
Homework Manager
Create assignments per session, review student submissions, give feedback
Progress Overview
Track plan completion %, homework submission rates, test scores
👨‍🎓 Student View
Today's Plan
AM/PM sessions, teacher, topic, what to prepare
Weekly Overview
This week's schedule with completion indicators
Homework & Submissions
Today's homework, submit photos/answers, track status
Quick Drill
20 repeat questions — flashcard style, track mastery
Resources & Tests
Study links per topic, upcoming tests, past scores
