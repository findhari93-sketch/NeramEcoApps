Key Behaviors

Pre-loaded options: When question list loads, ALL options for visible questions are fetched in background. Expanding a question shows options instantly — zero loading delay.
Virtual scrolling: All 624 questions render via virtual scroll (only ~20 in DOM at a time). Smooth scroll through the entire bank without performance issues.
Compact cards: Question cards use 6px gaps, 12px padding. No wasted space. Tags are tiny chips, not full badges. Text preview is 1-2 lines max.
Create Test flow: Toggle "Select" mode → check questions → "Create Test" → name it → done. Works with any filter combination. Filter first, then select.
Year Paper view: Clicking a year paper card opens the same question list, but pre-scoped to that paper's questions (e.g., JEE 2026 = 74 Qs). Same filter UI available to narrow further within the paper.
